# 💊 MedFind - Essential Medicine Availability Tracker

MedFind helps users locate nearby pharmacies with real-time stock updates for essential medicines like insulin and cardiac drugs.

## 🚀 Tech Stack
- HTML, CSS, JavaScript
- Tailwind CSS
- Leaflet + OpenStreetMap
- SweetAlert2
